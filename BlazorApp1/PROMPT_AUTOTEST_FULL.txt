13) Bo sung luong nghiep vu cap van hanh (bat buoc them)

AA. Session va dang nhap
- Session timeout: dang thao tac dang do thi het session => UI chuyen huong dang nhap, khong mat du lieu da luu
- Dang nhap sai lien tiep nhieu lan => thong bao dung, khong crash UI
- Dang nhap thanh cong o tab A, logout o tab B => tab A bi thu hoi session o request tiep theo
- Refresh token/session (neu co) hoat dong dung, khong tao loop redirect

AB. Da tab/da cua so

- Filter/sort tren tab A khong lam hong state tren tab B

AC. Khoi luong du lieu lon
